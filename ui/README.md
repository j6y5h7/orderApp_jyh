# 커피 주문 앱 - 프론트엔드 (UI)

React + Vite + JavaScript 기반 프론트엔드입니다.

## 요구 사항

- Node.js 18+
- npm 또는 yarn

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (기본: http://localhost:5173)
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview
```

## 폴더 구조

```
ui/
├── public/          # 정적 파일
├── src/
│   ├── App.jsx      # 루트 컴포넌트
│   ├── App.css
│   ├── main.jsx     # 진입점
│   └── index.css    # 전역 스타일
├── index.html
├── vite.config.js
└── package.json
```
